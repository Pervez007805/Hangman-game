# Hangman Game (Python)
This project is a **simple text-based Hangman game** developed in Python as part of my Python Programming.
- Language: Python
- Platform: Console-based

## Concepts Used
- random module  
- while loop  
- if-else conditions  
- strings & lists  

## Features
- Uses a predefined word list  
- Allows up to 6 incorrect guesses  
- Simple and interactive console gameplay
